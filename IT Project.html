My project name is knowledge base update.

In order to complete this ICT project, we need several qualified computers, developer websites and a qualified information management system.

In order to complete this scheme, we first need to collect some shortcomings of the old knowledge base. Based on the collected information, we need to develop a new knowledge base that can meet these needs. Secondly, after the development is completed, we need to test the knowledge base before the actual installation of the new knowledge base to eliminate some risk factors. After the test is completed, the actual installation can be carried out in the information management system.According to our prediction, after the actual installation, the new knowledge base will be more user-friendly and can accommodate more information.
